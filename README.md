# zblog

> 我的博客网站

> 技术栈：Vue + Vue-router + Vuex + axios + Element + node + mongodb

## Build Setup

``` bash

# 安装依赖
npm install

# 在本地localhost:8080服务器会以热加载的方式运行
npm run dev

# 压缩运代码
npm run build

# 构建运行代码并查看包分析器报告
npm run build --report
```
